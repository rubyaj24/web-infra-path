# Tool Logos Directory

This directory contains logos for all the tools and technologies referenced throughout the SaaS infrastructure learning path. Each logo should be in PNG format with transparent background and optimized for web display.

## Logo Specifications
- **Format**: PNG with transparent background
- **Size**: Recommended 32x32px or 64x64px for inline use
- **Naming**: Use lowercase with hyphens (kebab-case)

## Logo Sources

When adding new logos to this directory, you can source them from:

1. **Official Brand Assets**: Check the tool's official website for brand assets/press kits
2. **Simple Icons**: https://simpleicons.org/ - SVG icons for popular brands
3. **DevIcons**: https://devicon.dev/ - Programming languages and tools icons
4. **Logo Downloads**: https://logoeps.com/ - High-quality logo downloads
5. **Brand Guidelines**: Many companies provide official logos in their brand guidelines

## Usage Guidelines

- Always use official logos when available
- Respect brand guidelines and trademark policies
- Maintain consistent sizing across the documentation
- Use transparent backgrounds for better integration
- Optimize file sizes for web performance

## File Naming Convention

Logo files should follow this naming pattern:
- Use lowercase letters
- Replace spaces with hyphens
- Use descriptive names that match the tool name
- Example: `aws-lambda.png`, `react.png`, `styled-components.png`
